Network Controlled Rover with Home Automation & Surveillance System

  • Build Specifications:
  • Raspberry Pi 3B, Arduino Uno & Lilypad, HC-05 Bluetooth Module, PCA9685 I²C Servo Controller, Night-Vision RPi Camera, Ultrasonic and IR Sensors, Fire & Temperature Sensors, DC BO Motors & Wheels, L293D Motor Driver & Custom Power Board, Li-ion Battery Pack with BMS, XL4015E Step-Down Converter, 8-Channel Relay Module, USB Microphone, Custom Rover & Robotic Arm Chassis
    Project Description

    Built an IoT-based Remote Network Controlled Rover with integrated home automation and surveillance, enabling live video streaming, robotic arm control, and smart appliance management over Wi-Fi and cloud services.

    Developed using Raspberry Pi 3B with Python for ThingSpeak cloud communication and camera streaming, and Arduino Uno/Lilypad for motor, servo, and sensor operations. The system supports live video streaming via the Pi camera, with pan-tilt control through dual servo motors on a PCA9685 I²C controller. Navigation is enhanced using ultrasonic and IR sensors, while fire and temperature sensors improve safety. ThingSpeak is used for real-time data logging, remote command processing, and monitoring environmental readings.

    The project unifies remote rover control, video surveillance, and home automation into a modular system. It includes an 8-channel relay for smart appliance control, robust Li-ion battery power management, and a custom rover chassis, making it a complete IoT solution with cloud data access and live streaming capabilities.
