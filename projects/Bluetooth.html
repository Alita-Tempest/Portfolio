Bluetooth Multi Task Rover

  • Build Specifications:
  • Arduino Uno, HC-05 Bluetooth Module, Dual H-Bridge Motor Driver (L298N), DC Motors & Wheels, Ultrasonic Sensors, IR Line Sensors, 16×2 LCD Display (LiquidCrystal), Li-ion Battery Power Supply, Custom Rover Chassis
    Project Description

    Developed a versatile Bluetooth Multitask Rover that can be controlled via an Android app over a Bluetooth link, allowing manual drive commands, line-following, obstacle avoidance, and custom text display on an onboard LCD.

    Programmed in Arduino C++ for an Uno board, the system uses an HC-05 Bluetooth module to receive single-character commands, switches between line-follow and obstacle-avoidance modes using IR and ultrasonic sensors, and displays status or user-entered messages on a 16×2 LCD (LiquidCrystal). DC motors are driven through a dual H-bridge controller, while sensor readings on analog pins A0–A5 and digital pin 13 govern autonomous behaviors.

    This multitask rover provides smooth mode transitions, real-time sensor feedback, and an intuitive interface for text messaging. Its modular codebase and clear LCD prompts make it easy to extend or integrate new features.
